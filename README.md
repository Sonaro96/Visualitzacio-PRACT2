# Pràctica 2 - Visualitzacio de dades

## Autor

Roger Navas Solé

## Sobre el projecte

Aquest project integra dues bases de dades (trending_videos.csv i internet_access.csv) i les prepara per a una visualitzacio en Tableau, generant dos arxius (dades_netes_main i dades_netes_cat). 

## Com executar el codi

L'arxiu principal de codi és un R markdown file. S'ha generat amb R studio i s'han utilitzat llibreries actualitzades a data de maig de 2026.
